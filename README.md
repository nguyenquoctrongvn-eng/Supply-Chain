## Overview

This project analyzes a global supply chain dataset using SQL Server and Power BI to uncover insights related to sales performance, customer behavior, product profitability, and shipping efficiency. The dataset contains **65,536 rows and 53 columns**, providing detailed transactional and operational information across multiple business areas.

The dataset was obtained from [Kaggle](https://www.kaggle.com?utm_source=chatgpt.com) 

The products in the dataset belong to multiple categories such as:

* Clothing
* Sports
* Electronic Supplies

The SQL analysis includes:

* JOIN operations
* Common Table Expressions (CTEs)
* Window Functions
* Aggregations
* Ranking techniques
* KPI calculations

Key business metrics analyzed:

* Total Revenue
* Total Profit
* Customer Segmentation
* Product Performance
* Late Delivery Rate
* Discount Impact
* Geographic Sales Distribution

The processed data was connected to Power BI to create an interactive business intelligence dashboard featuring:

* Sales and profit overview
* Revenue trends over time
* Top-performing products
* Customer analysis
* Shipping and delivery insights
* Profitability analysis

Technologies used:

* Microsoft SQL Server
* Power BI
* SQL Server Management Studio (SSMS)
